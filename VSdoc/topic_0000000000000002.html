App Class

Namespace: Messenger
Assembly: Messenger (in Messenger.dll)
Class for the WPF application. This class is responsible for managing the application's lifecycle, including startup and exit events. It ensures that only a single instance of the application can run at a time by using a mutex. If another instance is already running, it brings the existing instance to the front and shuts down the new instance.
